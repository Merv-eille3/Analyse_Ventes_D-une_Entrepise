# Analyse_Ventes_D-une_Entrepise
Cette base de données est une base de données prise de Kaggle :
Dans ce répertoire, j'analyse les ventes d'une entreprise dont les clients sont particulièrement des personnes physiques.
Vous trouverez un scrit SQL, dont lequel ladite base de données est questionnée afin de répondre à certaines questions que pourraient se poser les reponsables de cette entreprise là.
De plus, vous verrez le script Python, un pdf et un powerpoint qui englobent les analyses et les recommandations sur lesquelles l'entreprise doit s'appuyer 
pour faire décoller les activités. Le rapport BI de cette base de données y est également.
